AI Habit Risk Predictor (CLI-Based ML Project)

Project Overview
The AI Habit Risk Predictor is a command-line based machine learning project that predicts whether a student is at risk of poor academic performance based on their daily habits.

The model analyzes three key factors:
- Sleep hours
- Study hours
- Screen time

Using these inputs, the system classifies the student into:
- Low Risk
- High Risk

Objectives
- Apply machine learning concepts in a real-life scenario
- Understand how lifestyle habits affect performance
- Build a simple predictive system using Logistic Regression

Features
- Takes real-time user input
- Uses a trained ML model for prediction
- Provides instant feedback
- Suggests improvements for better performance

Technologies Used
- Python 3
- Pandas (for data handling)
- Scikit-learn (for machine learning model)
